## How to Run Locally

Clone the repository:

```bash
  git clone
  https://github.com/rijviislam/assignment-2
```
Install dependencies:
```bash
  npm install
```
Create a .env file in the root:
```bash
  DATABASE_URL
  JWT_SECRET
  PORT
```
Start the server:
```bash
npm run dev
```
Production:
```bash
npm run build
npm start

```
